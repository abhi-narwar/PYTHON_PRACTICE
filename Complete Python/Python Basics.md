## 1. Introduction to Python

### What is Python?
- Python is a **high-level, interpreted, and object-oriented programming language**.
- It is easy to read and easy to write.

---

### Why Python is Popular?
- Simple and readable syntax
- Large number of libraries and frameworks
- Platform independent
- Used in many domains like AI, ML, Web, and Data Science

---

### Python Applications
- Web Development
- Data Analysis
- Machine Learning & AI
- Automation and Scripting
- Software Development
- Game Development

---

### Python vs Other Languages
- Python syntax is simpler than C, C++, and Java
- Python requires less code to perform tasks
- Python development is faster
- Python is beginner-friendly

---

## 2. Python Syntax & Basics

### Keywords
- Keywords are **reserved words** in Python.
- They have predefined meaning and cannot be used as variable names.

**Examples:** `if`, `else`, `for`, `while`, `True`, `False`

---

### Identifiers
- Identifiers are names used for variables, functions, or classes.
- They must start with a letter or underscore.
- They cannot start with a number.

**Example:** `name`, `_value`, `total_sum`

---

### Comments
- Comments are used to explain code.
- Python ignores comments during execution.

**Types:**
- Single-line comment: `#`
- Multi-line comment: `''' '''`

---

### Indentation
- Indentation refers to spaces at the beginning of a line.
- Python uses indentation to define blocks of code.
- Incorrect indentation causes errors.

---

### Variables
- Variables are used to store data.
- Python does not require data type declaration.

**Example:**  
`x = 10`

---

### Data Types Overview
- Data types define the type of data stored in a variable.

**Common Data Types:**
- int
- float
- string
- boolean
- complex

---

## Interview Conclusion
- Python is popular due to its simplicity and wide usage.
- Syntax is easy and readable.
- Indentation and keywords are very important in Python.
